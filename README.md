# SVM Kernel Tutorial

## Overview
This project demonstrates how different kernel functions affect the behaviour of Support Vector Machines (SVMs). The tutorial explores linear, polynomial, and RBF kernels, and analyses how they influence decision boundaries and model performance.

## Objectives
- Understand how SVM kernels work
- Compare kernel performance on different datasets
- Explore the effect of hyperparameters such as gamma

## Datasets
Two datasets were used:

1. Synthetic dataset: make_moons (from scikit-learn)
2. Real dataset: Iris dataset  
   https://www.kaggle.com/datasets/uciml/iris
